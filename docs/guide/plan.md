# 迭代计划

目前adminpro仍然处于开发阶段，计划会分出几个不同的版本分支进行维护，如下：

⭕：未开始

🪒：进行中

✅：已完成

- 🪒 简约版本：不包含多语言、权限功能、权限动态路由加载和Mock数据，只作为一个最基础的模板提供。
- ⭕ 前端完整版本：包含多语言、基础权限功能、动态路由加载和Mock数据，作为一个最基础的完整模板。
- ⭕ 前后端完整版：提供前端完整版本所有功能，并提供一套后台API的接口。
